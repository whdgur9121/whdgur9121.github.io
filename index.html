CSS의 기본 배경과 선택자

CSS는 홈페이지를 이루는 HTML과는 다른 문법이라고 할 수 있다. 보통 body가 아닌 head에 스타일을 삽입하거나, 불러오는 식으로 사용한다. 

        (head)
            (title)(/title)
            (style)
                h1 {
                    color : red;
                    background-color : orange;
                }
            (/style)
        (/head)

위의 상황에서 h1은 선택자이다. 선택자는 다음과 같다.

종류 형태
전체 선택자  *
태그 선택자 태그(ex h1, p)
아이디 선택자 #id
클래스 선택자 .class

태그선택자의 경우 body, p, h1 {margin:0} 처럼 사용이 가능하다.

아이디 선택자는 개별적인 하나의 것을 설정하기에 좋다. 아래는 활용이다.

(div id = "ex1")의 태그 부여 후 스타일에서 width 800px; margin : 0 auto; background-color : yello; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 width 800px; margin : 0 auto; overflow : hidden; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 color : white; width 200px; float : left; background: blue; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 color : white; width 600px; float : left; background: green; 적용

박스 속성

실제 눈에 보이는 것은 글자와 border(테두리)밖에 없다.

형태 설명
margin 속성 테두리와 다른 태그 사이의 테두리 바깥쪽 여백
border 속성 테두리
padding 속성 테두리와 글자 사이의 테두리 안쪽 여백, 배경은 패딩까지 적용
width 속성 글자를 감싸는 영역의 가로
height 속성 글자를 감싸는 영역의 세로

margin과 padding의 예시는 다음과 같다. 우측 위의 개발자도구를 활용하여 범위를 확인할 수 있다. 

    width: 100px; height: 100px;
    background-color: red;

    border: 20px solid black;
    margin: 10px; padding: 30px;

또한, margin(padding) : 0(위) 10(오른쪽) 0(아래) 10(왼쪽)으로 세부 설정도 가능하다.

박스 테두리의 경우는 다음과 같다.
div class=box로 태그의 클래스를 붙인 뒤, 스타일에서 적용한다.

.box{
굵기 : border-width : thick;(예시)
형태 : border-style : dashed;(예시)
색상 : border-color : black;(예시)
둥근 테두리 : border-radius : 20px;(예시) > 5px 10px 5px 10px로 왼쪽 위부터 설정도 가능하다.
}

물론 left나 right를 통해 개별 설정도 가능하다.
