CSS의 기본 배경과 선택자

CSS는 홈페이지를 이루는 HTML과는 다른 문법이라고 할 수 있다. 보통 body가 아닌 head에 스타일을 삽입하거나, 불러오는 식으로 사용한다. 

        (head)
            (title)(/title)
            (style)
                h1 {
                    color : red;
                    background-color : orange;
                }
            (/style)
        (/head)

위의 상황에서 h1은 선택자이다. 선택자는 다음과 같다.

종류 형태
전체 선택자  *
태그 선택자 태그(ex h1, p)
아이디 선택자 #id
클래스 선택자 .class

태그선택자의 경우 body, p, h1 {margin:0} 처럼 사용이 가능하다.

아이디 선택자는 개별적인 하나의 것을 설정하기에 좋다. 아래는 활용이다.

(div id = "ex1")의 태그 부여 후 스타일에서 width 800px; margin : 0 auto; background-color : yello; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 width 800px; margin : 0 auto; overflow : hidden; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 color : white; width 200px; float : left; background: blue; 적용
(div id = "ex2")의 태그 부여 후 스타일에서 color : white; width 600px; float : left; background: green; 적용
