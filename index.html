글자모양 태그

글자모양 태그는 헤드와 p를 감싸도 되고, 그 안에 필요한 만큼만 존재해도 된다.

굵은 글자 : (b)
기울어진 글자 : (i)
작은 글자 : (small)
아래첨자 : (sub)
위첨자 : (sup)
밑줄 글자 : (ins)
취소선 : (del)

목록 만들기

태그는 3가지이고 그 예시는 아래 제시한다.

(ul) : 순서가 없는 목록
(ol) : 순서가 있는 목록
(li) : 목록 요소

  • ui안에
  • li가
  • 각 구성요소로 존재한다.
  1. ui와 oi의 차이점은
  2. 번호이다.
중첩하면 다음과 같다.
  • li 하나를 먼저 넣고
    1. 목록 태그를 하나 더 넣고
    2. 똑같이 li를 통해 작성한다.
    3. 요약하면
    4. 목록태그1 > li > 목록태그2 > li 작성 > 목록태그2 닫기 > li닫기 > 목록태그 1 닫기이다. 

그 구조는 아래와 같다.
(ui)
    (li)
        (b) 굵은 글씨 범주 (/b)
        (ol)
            (li)작은 범주들(/li)
            (li)작은 범주들(/li)
            (li)작은 범주들(/li)
        (/ol)
    (/li)
(/ui)

표 : Table

기본적으로 표는 위에서부터 한줄한줄 적어간다고 생각해야 한다.
border : 표의 테두리 굵기
th, td colspan : 셀의 너비
th, td rowspan : 셀의 높이
자세한 내용은 2주차 강의노트를 참고하자.

태그 설명
tr 표에 행삽입
th 표의 제목셀 생성
tb 표의 일반셀 생성 

위의 표의 경우 다음과 같이 쓰여졌다.
(table border = "1")
    (thead)
        (tr)
            (th)태그(/th)
            (th)설명(/th)
        (/tr)
        (/thead)
    (tbody)
        (tr)
            (td)tr(/td)
            (td)표에 행삽입(/td)
        (/tr)
        (tr)
            (td)thead(/td)
            (td)표의 제목셀 생성(/td)
        (/tr)
        (tr)
            (td)tbody(/td)
            (td)표의 일반셀 생성(/td)
        (/tr)
    (/tbody)
(/table)

미디어 태그 : Img, audio, video

(img src =" " alt =" ", width=" ", height=" ")
앞에서부터 경로, 이미지 없을시 출력 문구, 너비, 높이이다.

(audio(video) controls="controls")
   (source src = " " type="audio/mp3")
(/audio(video))

위에서부터 컨트롤러 출력여부를 결정하고, 경로와 타입을 결정한다. 이후 태그를 끝낸다.
